# Testplan Opdracht 3.2: Netwerkconfiguratie_fysiek

Auteur(s) testplan: Jonas Feys

## voorbereiding

- Configurarie voor de swicth en router moet ingeladen zijn
- netwerkbekabeling
- actieve DC dient aanwezig te zijn, aangesloten via Bridged adapter
- Bij voorkeur een Director, cast en crew pc via Bridged adapter
- Een server uit het netwerk voor extra testen (trinity, neo, dozer of theoracle)


## testen

1. Krijgt de crew pc een IP adres in de correcte range?
2. Krijgt de cast pc een IP adres in de correcte range?
3. Kan je pingen van DC naar crew PC?
4. Kan je pingen van DC naar cast PC?
5. Kan je pingen van de extra server naar de DC?
5. Kan je pingen van crew PC naar de DC?
6. Heb je internet connectie op de DC?
7. Heb je internet connectie op de Crew/cast PC?
8. Kan je pingen van cast PC naar webserver? Indien nee => ok
9. Kan je pingen van crew PC naar webserver?
9. Kan je naar een webpagina surfen vanop de crew PC?
10. Kan je naar een webpagina surfen vanop de cast PC?
11. Kan je surfen naar http://www.thematrix.local?
12. Kan je surfen naar http://rallly.thematrix.local?
13. Kan je via RSAT vanop de director pc aan de DC?
